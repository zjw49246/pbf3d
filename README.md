# PBF3D
This project extends [taichi's pbf2d example](https://github.com/taichi-dev/taichi/blob/master/python/taichi/examples/simulation/pbf2d.py) to 3d.

## Usage
Create a Python virtual environment and install taichi
```angular2html
pip install taichi
```
Then run the code
```angular2html
python pbf3d.py
```
